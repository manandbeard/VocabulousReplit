# Vocabulous Docs

## MetaSRS Design Documents

These three files were copied verbatim from [`manandbeard/V3-Repo`](https://github.com/manandbeard/V3-Repo) and serve as the reference and design documentation for the MetaSRS / Vocabulous architecture. They capture the algorithm research, the agent build prompt, and the full engine source that underpin the spaced-repetition scheduling system being integrated into this project.

| Document | Description |
|----------|-------------|
| [meta-srs-research.md](meta-srs-research.md) | Algorithm design document covering the DSR memory model, Reptile meta-learning, FSRS-6 foundations, and scheduling math. |
| [meta-srs-agent-prompt.md](meta-srs-agent-prompt.md) | Full agent prompt used to scaffold the MetaSRS Python/PyTorch engine, including architecture specs, training loop, and deliverables. |
| [meta-srs-all-files.md](meta-srs-all-files.md) | Complete source code listing for all 31 files of the MetaSRS engine (config, models, training, inference, evaluation, and tests). |
